# rn-circular-progress

Circular progress bar for react native

## Installation

```sh
npm install rn-circular-progress
```

## Usage

```js
import { RnCircularProgressView } from "rn-circular-progress";

// ...

<RnCircularProgressView color="tomato" />
```
