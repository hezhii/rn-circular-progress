# rn-circular-progress

Circular progress bar for react native

## Installation

```sh
npm i react-native-svg
npm i rn-circular-progress
```

## Usage

```js
import CircularProgress from 'rn-circular-progress';

// ...

<CircularProgress progress={50} />;
```
